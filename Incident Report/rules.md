# 記入例

| 項目        | 内容                                           |
| --------- | -------------------------------------------- |
| **エラー名**  | ECONNREFUSED (接続拒否)                          |
| **発生環境**  | Node.js + PostgreSQL                         |
| **メッセージ** | `Error: connect ECONNREFUSED 127.0.0.1:5432` |
| **原因**    | DBコンテナがまだ起動していない状態でAPIが接続を試みた                |
| **解決方法**  | Docker Composeで`depends_on`を設定する             |
| **備考**    | 再現頻度：中 / チーム共有済み                             |
